### How to run:
For the UR5 Manipulator:
Be in this directory: `/cs593-rob-project`
and run: `python3 ur5.py`

For the Unitree A1 Robot:
Be in this directory: `/cs593-rob-project/unitree_pybullet`
and run: `python3 a1.py`
